üí† VPS DEPLOY 
- Get your [Necessary Variables](https://github.com/JIYOXC/AIU-USERBOT/blob/main/.env.sample)

- Clone the repository:    

  ‚Ä¢ `git clone https://github.com/JIYOXC/AIU-USERBOT`

- Go to the cloned folder:    

  ‚Ä¢ `cd AIU-USERBOT`

- Create a virtual env:      

  ‚Ä¢ `virtualenv -p /usr/bin/python3 venv`
    `. ./venv/bin/activate`

- Install the requirements:

  ‚Ä¢ `pip3 install --no-cache-dir  -r requirements.txt`

  ‚Ä¢ `pip3 install --no-cache-dir -r re*/st*/op*.txt`
  

- Fill your variables in the env by

  ‚Ä¢ `nano .env` 

  If you have finished edit, CTRL S + CTRL X.

‚Ä¢ Install screen to keep running your bot when you close the terminal by 
  
  ‚Ä¢ `screen -S AIU-USERBOT`

‚Ä¢ Finally Run the bot:
  
  ‚Ä¢ `bash startup`

‚Ä¢ For getting out from screen session press    
‚Ä¢ `Ctrl+a and Ctrl+d`
